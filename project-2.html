Project Overview

A website where you can search for two movies through inputs on the left and right side. The site then compares the stats of the movies (Awards, Revenue, Awards, Rating and Votes). The data about the movies comes from the OMDb API (Open Movie Database). It's created with VanillaJS the CSS library Bulma for styling.

Hosted on Netlify.
